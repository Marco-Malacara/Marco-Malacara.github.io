Titan

I project made for the music community!

Titan was designed to help manage events, specifically targeting the music industry. The application featured personalized profiles, instant messaging in real-time, budget handling, tour scheduling, event handling, and task managing between users.

  • • C# - Used for logic of application
  • • ASP.NET CORE – Used for architecture of application
  • • Azure App Services – Used for hosting and connection of local application to Azure cloud portal
  • • Azure SQL – Used for cloud database hosting
  • • Azure Storage Blob – Used for cloud storage of images and media
  • • Bluimp API – Used for imaging display on view pages
  • • Pusher API – Used for instant private and group messaging
  • • Bootstrap 4 – Used for polishing and modernizing website
  • Date: September 2018

eBarmony

A match site to find bars!

eBarmony was an application designed to simplify bar choices in a given area. The project allowed users to take a survey that used our self-made algorithm to match them to local bars. The user could also create bar crawls using the routes API, and review bars.

  • • C# - Used for logic of application
  • • ASP.NET CORE - Used for architecture of application
  • • SQL Database – Used for data storing and code first migrations
  • • Yelp Fusion API – Used for targeting local bars in a specified radius
  • • Yelp Review API – Used for grabbing yelp review and rating data to display along our data
  • • Google Routes API – Used for routing locations through google maps
  • • Bootstrap 4 – Used for polishing and modernizing website
  • Date: September 2018
